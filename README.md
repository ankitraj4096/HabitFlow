# HabitFlow: Smart Habit & Productivity Tracker 🚀

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Platform](https://img.shields.io/badge/Platform-Flutter%20%7C%20Android%20%7C%20iOS-blue)
![Status](https://img.shields.io/badge/Status-In%20Development-orange)

HabitFlow is an advanced, offline-first mobile application designed to empower users to build positive habits, manage their productivity intelligently, and engage with a supportive community. Built with Flutter, it offers a seamless and responsive experience across both Android and iOS devices.

The core of HabitFlow is its unique approach to productivity. It moves beyond simple task tracking by incorporating a personalized **Work-Rest Balance Timer**. During setup, users define their optimal work-to-rest ratio, and the app's main dashboard provides a real-time view of accumulated work and rest periods, promoting healthier and more sustainable productivity habits.

## ✨ Key Features

* **🧠 Dynamic Work-Rest Timer:** A personalized timer that helps users balance focused work sessions with scheduled rests for maximum efficiency.
* **✈️ Offline-First Architecture:** Guarantees 100% app functionality without an internet connection. Data is saved locally and synced seamlessly when connectivity is restored.
* **⏱️ Multi-Timer Support:** Gives users the flexibility to run multiple, concurrent timers for different habits or tasks simultaneously.
* **💬 Integrated Community & Sharing:** Features a real-time chat system. Users can share their habit progress and achievements directly with contacts or within community groups.
* **📊 Advanced Habit Management:** Includes daily habit renewal, detailed progress analytics, and the ability to hide completed tasks for a clean interface.
* **🔐 Secure Authentication:** Full user authentication to keep data private and synced across devices.

## 🛠️ Tech Stack & Architecture

* **Framework:** [Flutter](https://flutter.dev/)
* **Architecture:** Clean Architecture with Repository Pattern
* **State Management:** [Provider](https://pub.dev/packages/provider) with `ChangeNotifier`
* **Local Database:** [Drift (SQLite)](https://drift.simonbinder.eu/) for robust offline persistence
* **Backend & Sync:** [Firebase](https://firebase.google.com/) (Authentication, Firestore, Storage)
* **Language:** [Dart](https://dart.dev/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
* An editor like VS Code or Android Studio.
* A configured Firebase project.

### Installation

1. **Clone the repo**
git clone https://github.com/ankitraj4096/HabitFlow.git



2. **Navigate to the project directory**
cd habitflow


3. **Install dependencies**
flutter pub get


4. **Configure Firebase**
* Follow the instructions to add Firebase to your Flutter app using the [FlutterFire CLI](https://firebase.flutter.dev/docs/cli).
* Place your `google-services.json` (for Android) and `GoogleService-Info.plist` (for iOS) in the correct directories.

5. **Run the app**
flutter run
